# Homework task

Project created using ReactJs 17.0.1

- To prepare frontend environmet you have to run:
> `npm install`  
> `npm start`  

- you have to change your backend api url
> `/src/Utils/api.js`

-AddCampaigns method call from console:
> `AddCampaigns(data)`

- To create production build:
> `npm run build` 

- To run unit tests:
> `npm test`
